# Manual de Operações - Café da Júlio

## Guia para Baristas e Gerentes

Este manual explica como usar o painel administrativo do site do Café da Júlio para gerenciar o cardápio diário.

---

## 1. Acessar o Painel Administrativo

### Como Fazer Login:

1. Abra o site do Café da Júlio
2. Clique em **"Login Admin"** no canto superior direito
3. Use suas credenciais de administrador (fornecidas pelo gerente)
4. Você será redirecionado para o **Painel Administrativo**

### Tela de Boas-vindas:

Ao entrar, você verá uma mensagem de boas-vindas com seu nome e instruções básicas.

---

## 2. Gerenciar Disponibilidade de Itens

### Estrutura do Painel:

O painel possui três abas principais:

| Aba | Descrição |
|-----|-----------|
| **Disponíveis do Dia** | Itens especiais que variam diariamente |
| **Salgados** | Salgadinhos, pastéis, coxinhas, etc. |
| **Doces** | Bolos, biscoitos, sobremesas, etc. |

### Como Ativar/Desativar um Item:

1. Selecione a aba desejada (ex: "Disponíveis do Dia")
2. Você verá uma lista de todos os itens dessa categoria
3. Para cada item, há um **botão de alternância** (toggle switch)
4. Clique no botão para ativar (verde/disponível) ou desativar (cinza/indisponível)
5. A mudança é aplicada **imediatamente** no site

### Exemplo Prático:

```
Disponíveis do Dia (3 itens)

☑ Café Especial - R$ 8,50
   [Disponível] ← Clique aqui para desativar

☐ Bolo de Chocolate - R$ 12,00
   [Indisponível] ← Clique aqui para ativar

☑ Pão de Queijo - R$ 5,00
   [Disponível]
```

---

## 3. Rotina Diária Recomendada

### Manhã (Abertura):

1. Faça login no painel administrativo
2. Revise a aba **"Disponíveis do Dia"**
3. Ative os itens que foram preparados
4. Desative os itens que não estão prontos

### Durante o Dia:

1. Confira regularmente o painel
2. Quando um item acabar, **desative-o imediatamente**
3. Quando um novo item ficar pronto, **ative-o**

### Noite (Fechamento):

1. Desative todos os itens do dia
2. Verifique se todos os pedidos foram processados
3. Faça logout clicando em **"Sair"**

---

## 4. Gerenciamento de Marmitas Fit

### Categorias de Marmitas:

As marmitas fit estão organizadas em três categorias:

1. **Frango** - Opções com frango
2. **Carne/Suína** - Opções com carne bovina e suína
3. **Sopas** - Sopas nutritivas

### Como Gerenciar:

- O processo é o mesmo que para outros itens
- Use o painel para ativar/desativar marmitas
- Clientes verão apenas as marmitas disponíveis

---

## 5. Visualizar Estatísticas

### Informações Disponíveis:

Na aba de cada categoria, você verá:

- **Número total de itens**: Quantos itens existem naquela categoria
- **Itens disponíveis**: Quantos estão ativados no momento

### Exemplo:

```
Salgados (5 itens)
3 disponível(is)
```

Isso significa: 5 salgados no total, 3 estão ativados agora.

## 6. Receber Pedidos via WhatsApp

### Como Funcionam os Pedidos:

1. Cliente acessa o site
2. Seleciona itens disponíveis
3. Clica em **"Enviar via WhatsApp"**
4. Você recebe a mensagem com os detalhes do pedido

### Informações no Pedido:

```
*Novo Pedido - Café da Júlio*

*Cliente:* João Silva
*Telefone:* (54) 99999-9999

*Itens do Pedido:*
• Café Especial x2 - R$ 16,00
• Pão de Queijo x3 - R$ 15,00

*Total:* R$ 31,00

*Retirada:* Café da Júlio - Farroupilha
```

### Responder ao Cliente:

1. Confirme o recebimento
2. Informe o horário de retirada
3. Se necessário, peça esclarecimentos

## 7. Dicas Importantes

### ✅ Boas Práticas:

- **Atualize regularmente**: Desative itens assim que acabarem
- **Seja preciso**: Ative apenas o que está realmente disponível
- **Comunique**: Informe ao gerente sobre itens que acabam frequentemente
- **Faça backup**: Anote os itens mais vendidos

### ❌ Erros Comuns:

- Deixar itens desativados quando estão disponíveis
- Esquecer de desativar itens que acabaram
- Não responder aos pedidos rapidamente
- Desativar itens sem motivo

## 8. Troubleshooting

### Problema: "Não consigo fazer login"

**Solução:**
- Verifique suas credenciais
- Limpe o cache do navegador
- Tente em outro navegador
- Contacte o gerente

### Problema: "As mudanças não aparecem no site"

**Solução:**
- Atualize a página (F5 ou Ctrl+R)
- Aguarde alguns segundos
- Limpe o cache do navegador
- Verifique sua conexão de internet

### Problema: "Não recebi o pedido via WhatsApp"

**Solução:**
- Verifique se o WhatsApp está configurado corretamente
- Confirme que o número está correto
- Verifique as notificações do WhatsApp
- Contacte o gerente

## 12. Glossário

| Termo | Significado |
|-------|-------------|
| **Ativar** | Tornar um item visível e disponível para clientes |
| **Desativar** | Ocultar um item para que clientes não possam pedí-lo |
| **Toggle** | Botão de alternância que ativa/desativa |
| **Disponível** | Item que pode ser pedido pelos clientes |
| **Indisponível** | Item que não pode ser pedido |
| **Painel** | Interface de gerenciamento no site |
